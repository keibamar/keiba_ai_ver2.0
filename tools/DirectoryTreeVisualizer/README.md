# DirectoryTreeVisualizer
このプログラムは、指定されたディレクトリ内のファイルとフォルダを再帰的に探索し、ツリー形式で構造をテキストファイルに書き出します。

## 機能
指定されたディレクトリのフォルダ構成をツリー形式で表示

- .gitフォルダ、.gitignoreファイル、.csvファイル、.pickleファイル、.txtファイル、およびyyyymmdd形式の日付の名前を持つファイルやフォルダを表示から除外

- dataフォルダ内では、フォルダのみ表示

## 使い方
- このリポジトリをクローンまたはダウンロードします。

- プログラムを実行するためにPythonがインストールされていることを確認します。

## 実行方法
- ターミナルまたはコマンドプロンプトを開きます。

- プログラムのディレクトリに移動します。

- 以下のコマンドを実行します：

bash
python DirectoryTreeVisualizer.py
- プロンプトが表示されたら、スキャンしたいディレクトリのパスを入力します（未入力の場合は現在のディレクトリが使用されます）。

- DirectoryTree.txtという名前のファイルにディレクトリ構造が書き出されます。
